## About The Project

ETH my song allows fans and supportors to tip their favourite indie music artists.

## Tech stack and libraries
 - Solidity
 - Polygon Blockchain
 - Nextjs
 - web3.js
 - TailwindCSS
 - Metamask
 - IPFS
