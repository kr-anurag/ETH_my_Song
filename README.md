## About The Project

Donation dApp on Polygon to award your favorite content creator with MATIC tokens. 

## Tech stack and libraries
 - Solidity
 - Polygon Blockchain
 - Nextjs
 - web3.js
 - TailwindCSS
 - Metamask
 - IPFS
